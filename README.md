## [Overview](#overview)

leaf is a game / graphics framework written in C++.
It is designed with both the simplicity of a 2D game
and the complexity of a 3D game in mind.

The goal is to provide a simple standard library for game development 
that doesn't require the user to learn complex things but leaves the
door open. leaf provides an OpenGL like API in a C++ style. That means 
with more type saftey and using features like RAII and templates.

---

## [Installation](#installation)

To install leaf, all you have to do is clone the repository / fetch content 
via CMake and call add_subdirectory. leaf builds as a static library.
