# leaf glossary

#### General definitions

| Term   | Definition                                                                                                                                                                                               |
| ------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| extent | The spatial dimensions of an object in space, expressed along one or more axes (e.g., width, height, depth). Used for things that occupy or map to space, such as images, textures, or bounding volumes. |
| size   | The magnitude of an object, expressed as a single scalar value. The meaning depends on context: for example, the length of a vector or the memory footprint (in bytes) of a resource.                    |


#### Mirror terms

| Term A   | Term B  | Context                                                 |
| -------- | ------- | ------------------------------------------------------- |
| Build    | Reset   | construction and destruction of an object			   |
| Allocate | Release | management of resources for an object				   |
| Create   | Destroy | the combination of Build / Allocate and Reset / Release |
